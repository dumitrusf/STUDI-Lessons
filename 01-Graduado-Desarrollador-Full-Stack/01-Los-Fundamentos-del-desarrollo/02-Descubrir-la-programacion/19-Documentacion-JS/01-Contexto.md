# **Contexto**

<br>

---

<br>

El oficio de desarrollador requiere una buena capacidad para aprender por cuenta propia y para encontrar información en Internet.

De hecho, las tecnologías utilizadas son muy variadas y evolucionan rápidamente.

Por lo tanto, se necesita una buena autonomía para no quedarse bloqueado ante cualquier cambio en una tecnología.

Afortunadamente, Internet es una fuente inagotable de información, como lo demuestra el sitio web del Mozilla Developer Network (mozilla.org).

Este sitio alberga una documentación considerable, especialmente sobre JavaScript, que exploraremos en este módulo.
